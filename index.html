Fuel Economy Predictor

Alyx Gray

Pragmatic Problem Solving

Predicting vehicle fuel economy has practical application for different market sectors.

Prediction is done using a model built from a data collection of vehicle performance and build specifications. Stepwise selection for simple linear regression was used to build the model. This model uses vehicle weight, quarter-mile time, and transmission type as predictors for fuel efficiency.

## $call
## lm(formula = mpg ~ wt + qsec + am, data = mtcars)

## $coefficients
## (Intercept)          wt        qsec          am 
##    9.617781   -3.916504    1.225886    2.935837

User Interface

The application uses a Shiny web interface for receiving user input and reporting results.

Application: Buyer's Guide

The fuel economy predictor can be used to estimate the effect that transmission type will have on fuel efficiency for a vehicle of a specified weight with known performance characteristics. This knowledge can help a potential buyer decide what features he would like when optimizing his vehicle selection criteria.

We can predict the fuel economy for a vehicle that weighs 2000 pounds, has a manual transmission, and has a quarter-mile time of 15 seconds:

car_manual <- data.frame(wt = 2, qsec = 15, am = 1)
unname(predict(best_model, car_manual))

## [1] 23.1089

We can predict the fuel economy for the same vehicle that is equipped with an automatic transmission:

car_auto <- data.frame(wt = 2, qsec = 15, am = 0)
unname(predict(best_model, car_auto))

## [1] 20.17306

Application: High Performance Vehicles

The fuel economy predictor can be used to estimate fuel efficiency of high performance race car. For example, it can be used to estimate the effect of adding or removing weight from a vehicle on fuel consumption.

We can predict the fuel economy for a vehicle that weighs 2000 pounds, has a manual transmission, and has a quarter-mile time of 10 seconds:

racecar_unmodified <- data.frame(wt = 2, qsec = 10, am = 1)
unname(predict(best_model, racecar_unmodified))

## [1] 16.97947

We can predict the fuel economy for that same vehicle when the weight has been reduced to 1500 pounds:

racecar_modified <- data.frame(wt = 1.5, qsec = 10, am = 1)
unname(predict(best_model, racecar_modified))

## [1] 18.93772
